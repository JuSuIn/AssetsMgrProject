
## 🧮 A6. 자산 리밸런싱 제안

### 🎯 목적
현재 자산 구성과 목표 구성의 차이를 비교하여 리밸런싱 전략을 제안한다.

### 👥 참여 객체
| 객체 | 설명 |
|------|------|
| User | 리밸런싱 요청자 |
| React App | 리밸런싱 시뮬레이터 |
| Django API | 리밸런싱 요청 수신 |
| RebalanceService | 자산 비율 분석 로직 |
| PostgreSQL | 자산 데이터 |

### 🔄 시나리오 흐름
1. 사용자가 리밸런싱 메뉴 클릭
2. API에서 자산 현황 및 목표 구성 조회
3. RebalanceService에서 구성 차이 계산
4. 리밸런싱 비율/금액 계산
5. React에 추천 전략 표시
